Soroban abacus

you can use the up/down (▽/△) buttons on the abacus rows to modify the abacus values

background - each row in the abacus represents a digit; the top bead represents the value of 5 while each of the bottom beads represent a value of 1.
a number is counted when beads are positioned towards the middle beam (hide description)
